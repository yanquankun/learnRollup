# learnRollup
learn rollup.js and a simple project

### pre start

```shell
pnpm install
```

### start & watch

`typescript`
```shell
pnpm buildts
```

`javascript`
```shell
pnpm build
```

### build

`typescript`
1. `amd`
```shell
pnpm amdts
```
2. `cjs`
```shell
pnpm cjsts
```
3. `iife`
```shell
pnpm iifets
```
4. `es`
```shell
pnpm ests
```
5. `umd`
```shell
pnpm umdts
```
